AutoDok - windows приложение для автодокументирования програмного кода. Приложение анализирует написанные в исходниках комментарии и генерирует на их основе форматированные html документы.

Приложение состоит из нескольких частей:
1) AutoDokUI - это UI, который реализуется с помощью фреймворка Qt 5.x
2) AutoDokParser - это основной анализатор кода. Он реализуется отдельно от UI и должен иметь возможность запуска из командной строки. Для работы с регулярными выражениями используется библиотека boost::regex.
