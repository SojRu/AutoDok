AutoDok - windows приложение для автодокументирования програмного кода. Приложение анализирует написанные в исходниках коментарии и генерирует на их основе форматированные html документы.

Приложение состоит из нескольких частей:
1) UI реализуется с помощью фреймворка Qt 5.x
2) Парсер коментариев реализуется отдельно и должен иметь возможность запуска из командной строки.
